# Django Project

## 프로젝트 생성

- <b>Project Name</b> : mysite
- <b>App Name</b> : articles

## 페이지 생성

1. <b>index page</b>
   - `/index/` : 전체 게시글 목록을 보여줄 페이지
2. <b>create pages</b>
   - `/new/` : 글 작성을 위한 form(제목, 내용) 입력 페이지
   - `/create/` : 글 작성 결과(제목, 내용)를 출력하는 페이지

## 추가

1. <b>base template</b>
   - 모든 템플릿들이 상속받을 base.html
